                                                   💻 Technical Implementation
The app uses a specific algorithm to determine the relationship:

Character Stripping: Removes common letters between both names.

The Count: Sums the remaining unique characters.

The Culling: Loops through the "FLAMES" array, removing elements based on the count until only one remains.

The Reveal: Maps the final string to a specific image path in the assets folder.



                                                    🛠️ How to Run Locally
Clone this repository.

Open index.html in any modern web browser.
